# 🚀 MGPEB — Módulo de Gerenciamento de Pouso e Estabilização de Base

## 📖 Descrição do Projeto

Este projeto faz parte da **Missão Aurora Siger**, cujo objetivo é simular o estabelecimento da primeira colônia permanente em Marte.

O sistema desenvolvido, denominado **MGPEB (Módulo de Gerenciamento de Pouso e Estabilização de Base)**, é responsável por:

- Coordenar o pouso de módulos espaciais;
- Avaliar condições operacionais em tempo real;
- Organizar filas de pouso;
- Garantir segurança e eficiência na instalação da base.

---

## 🎯 Objetivo

Desenvolver um sistema que simule:

- A chegada de módulos à órbita marciana;
- A tomada de decisão para autorização de pouso;
- A organização dos módulos em diferentes estados (pousado, espera, alerta);
- A aplicação de lógica computacional e estruturas de dados.

---

## 🧠 Conceitos Aplicados

O projeto integra diversos conteúdos estudados:

- 🔌 **Portas lógicas e funções booleanas**
- 🐍 **Programação em Python**
- 📚 **Estruturas lineares (listas, filas e pilhas)**
- 🔍 **Algoritmos de busca e ordenação**
- 📐 **Modelagem de funções matemáticas**
- 🌱 **Princípios de governança (ESG)**
